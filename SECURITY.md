# Security Policy

## Supported Versions

Only the latest release receives security fixes.

| Version | Supported |
|---------|-----------|
| Latest  | ✅ |
| Older   | ❌ |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Send a report to [zakky1164@gmail.com](mailto:zakky1164@gmail.com) with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You can expect an acknowledgement within **3 business days** and a fix or mitigation plan within **14 days** depending on severity.

## Scope

ScrcpyDeck is a local network tool — it binds to `0.0.0.0` by default and has no authentication layer. It is intended to run in a trusted local network only. **Do not expose it to the public internet.**
